Loss Functions

Cross-entropy

$L$: Loss

$y$: Target value

$y’$: Prediction

Softmax

$L_i$: Loss for $i^{th}$ instance

$f_{y_i}$: Predicted class score of $x_i$ on the right class $y_i$

$f_j$: Predicted class score of $x_i$ on class $j$

Weight Decay

L2 Regularization

Regularized cost function

Gradient calculation

Gradient descent

$C$: Regularized cost function

$C_0$: Original cost function

$\eta$: Learning rate

L1 Regularization

Regularized cost function

Gradient calculation

Gradient descent

$\text{sgn}(x)$: signum function

Optimizers

Momentum

The momentum part is the related to the previous weight gradient which can be thought as adding effect of history.

AdaGrad

normalize the update for each weight

  • The problem is $S_i$ can never get smaller, which means the effective learning rates decreases monotonically decrease.

$S$: Sum squared gradient

$\epsilon$: A small constant to prevent dividing by 0 situation

RMSprop

AdaGrad forces the learning rate to always decrease, this constraint is relaxed with RMSProp

  • RMSProp can be viewed as a stochastic gradient descent version of RProp normalised by a moving average of the squared gradient

$\beta \approx 0.9$: the decay rate

Adam

“RMSProp with momentum”

  • The first order summed gradient is normalized by second order summed one

$M_i$ can be viewed as momentum part

$\alpha \approx 0.9$

$\beta \approx 0.999$
